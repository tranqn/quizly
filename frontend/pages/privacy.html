2. About this project

Quizly is a private portfolio/demo project (not a commercial service), built as part of a fullstack developer training program. This notice explains what happens to your data when you use it.

3. Hosting & server logs

This site runs on my own server. Every request is logged with technical access data (IP address, timestamp, requested page, browser type) for security and operational purposes only.

4. Account & authentication

Creating a demo account stores your email address and a hashed password. Login uses a JSON Web Token stored in an HttpOnly cookie, so the token is never exposed to client-side scripts. Legal basis: Art. 6(1)(b) GDPR (performance of the demo you request by signing up).

5. YouTube URLs & generated quizzes

When you submit a YouTube URL, the backend downloads the audio, transcribes it (Whisper) and generates quiz questions from the transcript using the Google Gemini API. The submitted URL and the resulting quiz are stored under your account so you can replay it later.

6. Retention & deletion

Demo accounts and generated quizzes are not kept indefinitely and can be deleted on request at any time by emailing the address above.

7. Your rights

You have the right to access, rectify, erase, and restrict processing of your personal data, and the right to lodge a complaint with a data protection supervisory authority. Contact the address above to exercise these rights.
